# Tween Learning

> Teaching Tech the African Way

A modern, responsive landing page for Tween Learning - a tech education platform offering live cohort-based programs in Data Science & AI, Cybersecurity, and Web Application Penetration Testing.

## 🚀 Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **Interactive Elements** - Smooth animations and transitions
- **Modern UI** - Clean, professional design with African-inspired branding
- **Functional Forms** - Enrollment, waitlist, and instructor application forms
- **Social Integration** - Connected social media links
- **Countdown Timer** - Dynamic countdown to next cohort launch

## 📚 Programs

### Data Science & AI
- 10 weeks intensive program
- Live online sessions
- Real-world projects
- Max 25 students per cohort

### Web Application Penetration Testing
- 8 weeks intensive program  
- Live online sessions
- Hands-on security training
- Max 20 students per cohort

## 🛠 Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern CSS with custom properties
- **Vanilla JavaScript** - No dependencies, pure JS functionality
- **Responsive Design** - Mobile-first approach

## 🎨 Design Features

- Custom CSS variables for consistent theming
- Smooth animations and micro-interactions
- Modal system for forms
- FAQ accordion
- Mobile hamburger menu
- Social media integration
